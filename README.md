## Technical Notes
- Animations are deliberately loopable and lightweight (capped particles, simple path math, requestAnimationFrame).
- Sound is 100% procedural (no audio files) — deep 48Hz rumble + bandpass crackle + high shimmer. Matches the white-light energy profile perfectly.
- The brick wall is high-detail CSS with staggered mortar, heavy vignette, concrete beams, and vertical metal security bars — remand-center institutional feel achieved without shipping large assets.
